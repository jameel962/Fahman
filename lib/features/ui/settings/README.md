# نظام الإعدادات - Settings System

## نظرة عامة
تم إنشاء نظام إعدادات شامل يتضمن إدارة الملف الشخصي والإشعارات واللغة.

## الملفات المُنشأة

### الصفحات (Screens)
1. **`profile_screen.dart`** - صفحة الملف الشخصي الرئيسية
   - عرض صورة المستخدم والاسم والبريد الإلكتروني
   - زر تعديل للانتقال إلى صفحة التعديل
   - أقسام الإعدادات (الإشعارات، اللغة، كلمة المرور)
   - قسم "حول" مع معلومات التطبيق

2. **`edit_profile_screen.dart`** - صفحة تعديل الملف الشخصي
   - إمكانية تغيير صورة المستخدم
   - تعديل الاسم والبريد الإلكتروني
   - زر حفظ للتأكيد

3. **`change_password_screen.dart`** - صفحة تغيير كلمة المرور
   - إدخال كلمة المرور الحالية
   - إدخال كلمة المرور الجديدة
   - تأكيد كلمة المرور الجديدة
   - التحقق من صحة البيانات

4. **`settings_screen.dart`** - صفحة الإعدادات العامة
   - إدارة الإشعارات (مفتاح/مغلق)
   - اختيار اللغة (عربي/إنجليزي)
   - إعدادات الأمان

### المكونات (Widgets)
1. **`settings_app_bar.dart`** - شريط التطبيق للإعدادات
   - تصميم موحد لجميع صفحات الإعدادات
   - زر العودة مع العنوان

2. **`language_selector.dart`** - مكون اختيار اللغة
   - قائمة منسدلة لاختيار اللغة
   - عرض اللغات المتاحة مع الأيقونات
   - ربط مع نظام الترجمة

## المسارات (Routes)
تم إضافة المسارات التالية في `routes.dart`:
- `/profile` - صفحة الملف الشخصي
- `/edit-profile` - صفحة تعديل الملف الشخصي
- `/change-password` - صفحة تغيير كلمة المرور
- `/settings` - صفحة الإعدادات

## التكامل مع النظام
- تم ربط زر الإعدادات في الصفحة الرئيسية بصفحة الملف الشخصي
- تم ربط زر الإشعارات في الصفحة الرئيسية بصفحة الإعدادات
- تم إزالة زر اللغة من الصفحة الرئيسية وربطه بنظام الإعدادات
- تم ربط مبدل اللغة بنظام الترجمة `easy_localization`

## الاستخدام
1. من الصفحة الرئيسية، اضغط على أيقونة الإعدادات للوصول إلى الملف الشخصي
2. اضغط على "تعديل" لتعديل البيانات الشخصية
3. اضغط على "كلمة المرور" لتغيير كلمة المرور
4. اضغط على "الإشعارات" لإدارة الإشعارات واللغة
